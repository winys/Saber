### How to run it?
1. Install packages
    ```
    npm install
    cd app
    npm install
    ```

2. Run on dev model
    ```
    npm run dev
    node_modules/electron-prebuilt/dist/electron.exe main.js
    ```
> Tips: If you use vscode to run it, you could just press #F5# to run it.
